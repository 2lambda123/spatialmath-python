se(3) twist¶

TWIST SE(2) and SE(3) Twist class

A Twist class holds the parameters of a twist, a representation of a rigid body displacement in SE(2) or SE(3).

A¶

Array value of an instance (SMUserList superclass method)

Returns

NumPy array value of this instance

Return type

ndarray

  • X.A is a NumPy array that represents the value of this instance, and has a shape given by X.shape.

Note

This assumes that len(X) == 1, ie. it is a single-valued instance.

N¶

Dimension of the object’s vector representation

Returns

dimension

Return type

int

Length of the Twist vector

Example:

>>> x = Twist()
>>> x.N
3
S¶

Twist as a vector (superclass property)

Returns

Twist vector

Return type

numpy.ndarray, shape=(N,)

  • X.S is a 3-vector if X is a Twist2 instance, and a 6-vector if X is a Twist instance.

isprismatic¶

Test for prismatic twist (superclass property)

Returns

Whether twist is purely prismatic

Return type

bool

Example:

>>> x = Twist3.R([1,2,3], [4,5,6])
>>> x.isprismatic
False
isunit¶

Test for unit twist (superclass property)

Returns

If twist is a unit-twist

Return type

bool

shape¶

Shape of the object’s interal matrix representation

Returns

(6,)

Return type

tuple

unit¶

Unitize twist (superclass property)

Returns

a unit twist

Return type

Twist3 or Twist2

twist.unit() is a Twist object representing a unit aligned with the Twist twist.

v¶

Moment vector of twist

Returns

Moment vector

Return type

numpy.ndarray, shape=(3,)

X.v is a 3-vector representing the moment vector of the twist.

For example:

>>> t = Twist3([1, 2, 3, 4, 5, 6])
>>> t.v
array([1, 2, 3])
w¶

Direction vector of twist

Returns

Direction vector

Return type

numpy.ndarray, shape=(3,)

X.w is a 3-vector representing the direction vector of the twist.

For example:

>>> t = Twist3([1, 2, 3, 4, 5, 6])
>>> t.w
array([4, 5, 6])
